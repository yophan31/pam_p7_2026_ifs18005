# Delcom Plants dengan Flutter

Menampilkan berbagai daftar tumbuhan.

# Install Package

```bash
flutter pub get

flutter pub run flutter_launcher_icons

dart run launcher_name:main
```

# Melakukan Pengujian

```bash
# Melakukan keseluruhan pengujian
flutter test

# Melakukan pengujian per folder
flutter test test/unit/
flutter test test/widget/
flutter test test/screens/
flutter test test/integration/
```
